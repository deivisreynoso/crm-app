## Who these manuals are for

- **Owner / Admin** — full setup, team, integrations, automations, finances configuration
- **Sales** — full CRM write (contacts, quotes, invoices, project stages, Drive); no deletes, expenses, or admin settings
- **Viewer** — read-only demo access; callouts note what is blocked

## CRM UI language

The CRM interface follows **Settings → Platform language** (`English` / `Spanish`). Customer-facing emails and public pages use each contact's **Preferred language** field, not the workspace language.
